## Architectural Gotchas

### Server dist gate

New `server/routes/*.ts` files won't be served until compiled ‚Äî the server runs from `server/dist/`. A 404 on a new route almost always means it's missing from `server/dist/routes/`. Fix: `npx tsc --project ./server/tsconfig.json`. Check `ls server/dist/routes/` first before debugging anything else.

### Two Tailwind instances in microsites

Microsites inject Tailwind Play CDN via `useEffect`. The CDN ships its own preflight CSS that resets `border-width: 0`, which conflicts with the compiled project Tailwind. Tailwind border classes (`border border-white`) silently fail. Use inline `style={{ border: '1px solid white' }}` for any border in microsite components.

### Browser default link color overrides inherited color

`<a>` elements have a hardcoded UA stylesheet `color: blue` that wins over `color: inherit`. Add `a { color: inherit; }` to `CUSTOM_STYLES` in microsite components ‚Äî even if the parent has an explicit color set.

### `Math.random()` in render causes re-render loops

Calling `Math.random()` at component level (e.g. to pick a random background image) re-runs on every state change ‚Äî typing one character in a textarea triggers the component to pick a new image. Stabilize: `const bg = useMemo(() => IMAGES[Math.floor(Math.random() * IMAGES.length)], [])`. Zero deps = runs once on mount.

### Hover-pause fires immediately on large components

`onMouseEnter ‚Üí setPaused(true)` on a full-screen (85vh+) component fires the instant the page loads because the cursor is already sitting over it. Autoplay starts, then immediately pauses. Use click-to-toggle for pause ‚Äî `onClick={() => setPaused(p => !p)}`.

### `users.id` is `bigint`, not uuid

The `users` table has `id bigint generated by default as identity`. Any migration with a FK to `users(id)` must use `bigint`, not `uuid`. Also: no `name` column ‚Äî use `trim(first_name || ' ' || last_name)`.

### Mailgun `from` must be an authorized sending domain

Setting `from: 'Name <gmail@gmail.com>'` violates Gmail's DMARC policy ‚Äî Mailgun rejects it silently (email never arrives). Always use `EMAIL_CONFIG.from` (the Mailgun-authorized domain) as the sender. Set `replyTo: userEmail` so replies go to the right person.

### Portfolio JSON must be a single valid array

Webpack parses `portfolio.json` at build time. If a second array was appended after the closing `]` (e.g. from a partial edit), webpack throws a silent parse error and the component renders nothing. Always rewrite the entire file rather than appending.

### Config.ts is the single source of truth for Portfolio

Never hardcode `templateId`, `autoplay`, or `showNav` in JSX. Read from `config.ts`. The config file doesn't always trigger HMR ‚Äî if changes aren't reflected, touch the consumer file (`src/sites/moxilabs/index.tsx`) or restart the dev server.
